# OmniMind-Agent 测试用例文档

## 1. 测试概述

本测试文档定义了 OmniMind-Agent 的分层测试策略，包括单元测试、集成测试、端到端测试和性能测试。

---

## 2. 测试环境

| 环境类型 | 描述 |
| :--- | :--- |
| **开发环境** | 本地开发机，用于单元测试 |
| **测试环境** | 隔离的测试服务器，用于集成测试 |
| **生产环境** | 生产服务器，用于性能和回归测试 |

| **UT-LEARN-011** | 闪卡生成：空白笔记 | 返回错误，提示无有效内容 |
| **UT-LEARN-012** | 闪卡生成：非文本内容（纯代码文件） | 返回错误或提示内容不适用 |
| **UT-LEARN-013** | 思维导图生成：20 个闪卡输入 | 生成 Mermaid 格式，至少 3 个一级节点 |
| **UT-LEARN-014** | 思维导图生成：空闪卡列表 | 返回错误，提示无可用知识点 |
| **UT-LEARN-015** | 学习报告生成：1 周数据 | 统计数据准确（任务完成数、学习时长、掌握度） |
| **UT-LEARN-016** | 薄弱点识别：多次测验同一概念答错 | 该概念 mastery < 0.3，出现在 weak_points 列表中 |

| 测试用例 | 场景 | 预期结果 |
